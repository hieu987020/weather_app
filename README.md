# weather_app

A new Weather project.

# Install
1. Open terminal at folder you want to clone project
2. Clone project
```
git clone https://github.com/hieu987020/weather_app
```
3. Open project folder
```
cd weather_app
```
4. Get Flutter packages
```
flutter pub get
```
5. Run project
```
flutter run
```
